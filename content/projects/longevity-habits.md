## 关于这个项目

这是一个帮助用户建立和追踪健康习惯的工具。基于长寿研究，提供科学的习惯建议和可视化的进度追踪。

**核心功能**：
- 科学的习惯模板（基于长寿研究）
- 每日打卡与进度追踪
- 数据可视化与洞察
- 本地存储，保护隐私

## 技术栈

- React + TypeScript
- Vite 构建
- Tailwind CSS 样式
- Zustand 状态管理
- Framer Motion 动画

## 我学到什么

1. **状态管理选择**：Zustand 比 Redux 更轻量，对于中小型项目非常合适
2. **动画克制**：Framer Motion 功能强大，但动画应该服务于体验而非炫技
3. **习惯产品核心**：降低记录摩擦比功能丰富更重要
