# 前置准备

在开始之前，需要提前准备好三样东西：  
- [ ] 油猴插件
- [ ] 可用脚本
- [ ] 多线程下载工具

浏览器全程打开开发者模式，一般在拓展页面设置：
![[Pasted image 20260122170929.png]]
![[Pasted image 20260122171005.png]]


整个流程并不复杂，但每一步都需要正确配置，否则很容易出现“脚本装了但没效果”的情况。

---

## 一、篡改猴（Tampermonkey）工具

篡改猴是一款浏览器脚本管理工具，用来运行各类用户脚本（通常称为“油猴脚本”）。

在常见浏览器的插件商店中，直接搜索“篡改猴”即可找到。

目前主流的油猴插件主要有三种：

Google Chrome 浏览器中的“黑猴”  
Microsoft Edge 浏览器中的“红猴”和“绿猴”

![[Pasted image 20260122145742.png]]
![[Pasted image 20260122145818.png]]

我个人目前主要使用的是 **Edge 浏览器中的红猴（测试版）**。  
该版本在脚本兼容性和性能方面表现较好，对部分网盘脚本的支持也更稳定。

安装完成后，浏览器右上角会出现一个猴子图标，说明插件已准备就绪。

点击插件，进入详情页面，允许脚本运行及访问所有网页。
![[Pasted image 20260122171351.png]]
![[Pasted image 20260122171218.png]]

## 二、脚本来源与安装

市面上的脚本管理与分享网站不少，直接搜索“脚本管理网站”即可找到。

我主要使用的是 Tampermonkey 官方脚本库：
```
https://www.tampermonkey.net
```

进入网站后，选择你当前使用的浏览器，页面会提供一份简要的新手使用说明。

![[Pasted image 20260122150527.png]]

点击「用户脚本」，即可进入脚本搜索页面。  
该页面汇集了大量可一键安装的脚本，涵盖网盘、网页增强、视频解析等常见需求。

![[Pasted image 20260122150709.png]]

以**夸克网盘**为例，在搜索框中输入“夸克网盘”，即可看到大量相关脚本。

![[Pasted image 20260122151650.png]]

下面以 **LinkSwift** 为例说明安装流程：

进入脚本页面后，点击项目链接，通常会跳转到 GitHub  
![[Pasted image 20260122151818.png]]

在页面中找到脚本的安装链接  ，点击后浏览器会自动跳转到 Tampermonkey 的脚本安装页面：

![[Pasted image 20260122151946.png]]

点击“安装”即可完成脚本添加。

安装完成后，点击浏览器右上角的篡改猴图标  
进入管理面板，确认对应脚本已启用（开关为开启状态）。

## 三、下载工具准备

脚本的作用是 **解析并获取真实下载链接**，并不直接负责高速下载。  
因此需要提前准备一个支持多线程的下载器。

常见的下载工具包括 IDM、XDown、Motrix  等。

我个人更常使用 **Motrix**，其基于 Aria2，支持多线程下载，跨平台，配置相对简单。

在浏览器中搜索 Motrix，下载安装即可。  
![[Pasted image 20260122152534.png]]

安装完成后，进入设置，更改以下内容：
- 默认下载路径（可选）
- 传输设置：限速处填写任意大于65535的数字取得最大限速，并更改下载速度单位为Mb/s
- 开启持续做种
- 进行任务管理，自定义最大同时下载数量，并将服务器最大连接数调至最大
更改完成后点击”保存并应用“即可

![[Pasted image 20260122172207.png]]

## 四、测试脚本并获取下载链接

完成上述准备后，可以开始测试脚本是否生效。

首先点击浏览器右上角的篡改猴图标，确认目标脚本处于“启用”状态
![[Pasted image 20260122152717.png]]

随后进入网盘的文件分享页面，如果脚本生效，页面中通常会出现“下载助手”“高速下载”“解析下载”等按钮，按钮位置会因脚本类型和网盘页面版本有所不同。

![[Pasted image 20260122152933.png]]

点击下载助手后，脚本一般会提供多种导出方式，例如：

- Aria2  
- cURL  
- API  

不同导出方式需要不同的下载器，而 Motrix 对应Aria2。

![[Pasted image 20260122164025.png]]

此处选择 **Aria2**。

进入推送页面后，点击“全部推送至下载器”  

![[Pasted image 20260122153347.png]]

若 Motrix 正常运行，会弹出推送成功提示

![[Pasted image 20260122153546.png]]

随后切换到下载器界面  
可以看到多个任务同时开始下载，下载速度明显高于网页端限速

![[Pasted image 20260122153724.png]]

至此即可确认：  
脚本解析成功，下载器已成功接管任务，网盘限速被绕过。

## 五、常见问题提醒

如果脚本未生效，通常不是脚本本身的问题，而是以下原因之一：

- 脚本未启用  
- 脚本与当前网盘页面版本不兼容  
- 下载器未运行或未正确关联  
- 浏览器的隐私或安全插件拦截了脚本请求  

遇到问题时，建议优先更换脚本版本，或查看脚本页面中的说明与评论区反馈。
